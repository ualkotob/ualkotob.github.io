Variables are data and, in JavaScript, functions can be stored in variables. This means we can treat them like other variables.

Passing through Functions

The add variable is also a function. Here we can see that means we can pass it in as a parameter of the useNumbers function. Functions passed this way are called callback functions. See my callback functions tutorial for more information.

Storing Functions in Arrays

In this example we store four functions in our arithmetic array. They are called like you would normally call array elements using arithmetic[#]. This can be useful for storing functions together, or performing a string of actions based on the array of functions.

Conclusion

Since functions work differently in JavaScript, people coming from different programming languages need to adjust. Functions can be treated like other variables, and this opens up a lot of new possibilities for programming.
