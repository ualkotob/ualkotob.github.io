Inheritance in JavaScript

In JavaScript, inheritance is when one object inherits the properties/methods of another object. We will call is a child parent relationship. The child object takes all of the properties and methods of the parent object, unless it already has them. See the example below.

Classical Inheritance vs Prototypal Inheritance

The paradigm of inheriting from objects is fundamentally different from inheriting from classes. We can think of classes as more of a template for creating objects. They do not function on their own, and cannot be invoked without being instantiated in an object. When we use objects for inheritance, the parent object is always accessible directly, and the values can be changed after runtime.
