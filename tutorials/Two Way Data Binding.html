Two way data binding is when you have an element in the UI that is updated by an object in the script. Then, when the UI is updated, the object is also updated.

AngularJS Example

In AngularJS, two way databinding is a built in feature. In the example above we created the testData variable in the scope. We then used ng-model to bind it to the input box. This makes it bound to the variable, and change it when we type something in the box. To make sure this is a two way bind, we display testData below the box as well using the angular expression {{testData}}. It also updates when we change the test in the input box. As we can see, it is working as intended.

Conclusion

Two way data binding is a useful tool. By using it correctly, we can create more responsive and user friendly web applications.
