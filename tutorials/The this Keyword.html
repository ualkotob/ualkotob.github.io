The this keyword is an important tool in developing JavaScript functions. It refers to the owner of the function being executed.

Object Owner and Inheritance

In this example we use this.firstName in the method greeting to refer to the firstName property in the person object. this refers to person, as it is the current owner of the function greeting. Now, lets see what happens if the method is inherited.

In this example, we can see when this refers to student. Even though the greeting method was inherited from person. Since it is being called from student, student is the owner of greeting in this instance.

Conclusion

The this keyword is a fairly simple to use and powerful tool. When using it we just need to remember the owner of that function.
